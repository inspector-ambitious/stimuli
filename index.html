Domino

A Dom Events injection library.

Download latest stable version

Supported Browsers:

  • IE8-10
  • Firefox
  • Chrome
  • Safari
  • PhantomJS

How to build

Install grunt:

npm install -g grunt-cli

Install all required node modules: 

npm install

Build domino in build/domino.js:

grunt build

How to tests

To lint all js files:

grunt jshint

To execute all unit tests, use:

grunt test

To generate test coverage report in coverage/ folder, use:

grunt coverage

Contributors Tips

  • You may want to have a look at the generated JSDoc

  • Please send a pull request to the development branch
